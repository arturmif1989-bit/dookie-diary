# Dookie Diary 💩

Карта твоих подвигов. Регистрируйся, отмечай места, добавляй друзей.

## Стек

- Чистый HTML/CSS/JS, без фреймворков
- [Leaflet](https://leafletjs.com/) для карты (бесплатно, OpenStreetMap)
- [Supabase](https://supabase.com) для базы данных и аутентификации
- Hosted on [Vercel](https://vercel.com)

## Структура

- `index.html` — главная страница
- `styles.css` — стили
- `app.js` — вся логика
